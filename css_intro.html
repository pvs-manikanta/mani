CSS Introduction (పరిచయం)

CSS అంటే Cascading Style Sheets. HTML తో మనం క్రియేట్ చేసిన వెబ్ పేజీలకి రంగులు (Colors), అందమైన లేఅవుట్స్, డిజైన్స్ అప్లై చేయడానికి CSS వాడుతాం.

ఒక చిన్న ఉదాహరణతో చెప్పాలంటే: HTML అనేది మనిషి అస్థిపంజరం అయితే, CSS అనేది మనిషి వేసుకునే అందమైన బట్టలు మరియు మేకప్ లాంటిది!

CSS Syntax (రాసే విధానం)

CSS రాయడానికి ఒక Selector, Property మరియు Value ఉండాలి.

CSS Example:

h1 {
    color: blue;
    font-size: 20px;
    text-align: center;
}
