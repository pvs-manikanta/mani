CSS Introduction

It is the standard stylesheet language used to design, style, and lay out web pages.

CSS Syntax

A CSS rule consists of a selector and a declaration block.

  • selector
  • declaration
  • property
  • value

CSS Example:

h1 {
    color: blue;
    font-size: 20px;
    text-align: center;
}
