HTML Introduction (పరిచయం)

HTML అంటే HyperText Markup Language. ఒక వెబ్‌సైట్ యొక్క అస్థిపంజరం (Structure) క్రియేట్ చేయడానికి HTML వాడుతాం.

నేను నేర్చుకున్న దాని ప్రకారం, ఒక వెబ్ పేజీ బిల్డ్ అవ్వడానికి HTML చాలా ముఖ్యం. కింద దీని బేసిక్ కోడ్ చూడండి:

ఉదాహరణ (Example):

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
