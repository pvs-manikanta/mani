HTML Basic Tags (ముఖ్యమైన టాగులు)

HTML లో మనం కంటెంట్‌ను చూపించడానికి రకరకాల టాగులు వాడుతాం. వాటిలో కొన్ని ముఖ్యమైనవి ఇక్కడ చూద్దాం:

  • <h1> to <h6>: ఇవి హెడ్డింగ్స్ (Headings) కోసం వాడుతాం. <h1> చాలా పెద్దదిగా, <h6> చాలా చిన్నదిగా ఉంటుంది.
  • <p>: పారాగ్రాఫ్ (Paragraph) రాయడానికి వాడుతాం.
  • <b> లేదా <strong>: టెక్స్ట్‌ని బోల్డ్ (వుబ్బెత్తుగా) చేయడానికి వాడుతాం.

Tags Example:

<h1>ఇది మెయిన్ హెడ్డింగ్</h1>
<p>ఇది ఒక పారాగ్రాఫ్ టెక్స్ట్.</p>
<p>ఇందులో కొన్ని ముఖ్యాంశాలు <strong>బోల్డ్</strong> గా ఉన్నాయి.</p>
